# Data Sources

## Daily Data

- `nifty50.csv` — NIFTY 50 daily OHLCV from Yahoo Finance (`^NSEI`), 2007-2026
- `indiavix.csv` — India VIX daily close from Yahoo Finance (`^INDIAVIX`), 2007-2026

Downloaded via `yfinance` in `main.py`.

## Intraday Data

- `NIFTY 50_5minute.csv` — NIFTY 50 5-minute OHLCV candles, 2015-01-09 to 2026-01-22

Source: [Kaggle — NIFTY 50 Minute Data](https://www.kaggle.com/datasets/debashis74017/nifty-50-minute-data)

Columns: `date, open, high, low, close, volume`
